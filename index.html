My Second Project with Claude

Create my personal CV website

Challenge

Use Claude in a better way than the previous project to create my personal CV website. Claude in the browser is tedious and error prone. In the previous project i had to start over many times.

Solution

Learned about Claude Code! Claude Code is a terminal-based, AI-powered agentic coding tool that helps developers analyze, plan, write, and edit code. It integrates with your IDE and is installed with a node package and runs in your terminal. I use Visual studio code with a claude extension and commit my work to my GitHub repository. Then i asked claude to read my cv pdf file exported from LinkedIn and create my cv website with the design based on another website.

Outcome

Claude made an my CV webpage, perfectly reflecting the details in my CV pdf file. It made the design based on the website i referenced. I had to ask it to fix some bugs and make some changes, but overall it was a great experience and I learned a lot about how to use Claude Code.

Lessons Learned

1

Claude code understands your code base

Using Claude code is a game changer. It understands the code. Integrating with your IDE and is installed with a node package and runs in your terminal.

2

Incremental changes

Asking Claude to make small changes and commit them to GitHub is a great way to work. It makes it easy to track changes and revert if needed.

3

You need to have a Claude PRO subscription to get the most out of it

I had to upgrade to Claude Pro to use Claude Code. It is worth it. Only €15 gives you 5 times more tokens than the free version. (tokens are used to process your requests, every 5 hours you get a token refresh)

4

Code Review Partner

Claude excels as a code review partner, catching issues and suggesting improvements.
