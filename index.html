My Second Project with Claude

Create my personal CV website

Challenge

Use Claude in a better way than the previous project to create my personal CV website. Claude in the browser is tedious and error prone. In the previous project i had to start over many times.

Solution

Learned about Claude Code! Claude Code is a terminal-based, AI-powered agentic coding tool that helps developers analyze, plan, write, and edit code. It integrates with your IDE and is installed with a node package and runs in your terminal. I use Visual studio code with a Claude extension and commit my work to my GitHub repository. Then i asked Claude to read my cv pdf file exported from LinkedIn and create my cv website with the design based on another website.

Outcome

Claude made an my CV webpage, perfectly reflecting the details in my CV pdf file. It made the design based on the website i referenced. I had to ask it to fix some bugs and make some changes, but overall it was a great experience and I learned a lot about how to use Claude Code.

Lessons Learned

1

Claude code understands your code base

Using Claude code is a game changer. It understands the code. Integrating with your IDE and is installed with a node package and runs in your terminal.

2

Incremental changes

Asking Claude to make small changes and commit them to GitHub is a great way to work. It makes it easy to track changes and revert if needed.

3

You need to have a Claude PRO subscription to get the most out of it

I had to upgrade to Claude Pro to use Claude Code. It is worth it. Only €15 gives you 5 times more tokens than the free version. (tokens are used to process your requests, every 5 hours you get a token refresh)

4

Code Review Partner

Claude excels as a code review partner, catching issues and suggesting improvements.

Lessons Learned

1

Claude Pro subscription is not enough if you use it a lot!

When you use Claude code, you have a certain amount of tokens you can use per 5 hours. I you use your tokens up, you have to wait until they refresh. This is very frustrating when you are in the middle of a project. I also found out that only have access to the Sonnet 4 model (for everyday use) and not the more advanced Opus 4 modul (for complex tasks). I had to upgrade to a max plan to get access to Opus 4. The Pro subscription is €15 per month, the Max subscription is €90 per month.

2

Claude is a great pair programming partner, but he needs you!

It is great that Claude can make a lot of the code for you, but you have to be there to guide him. You save all the time you would have spend researching and writing boilerplate code. I learned a lot about how to debug together with Claude, if you know how to debug and give him clues he can solve complex problems quickly. Asking him to solve a problem again and again will make him try different solutions until the problem is solved, but sometimes he gets stuck in a loop and you have to help him out.

3

Ask Claude to organise your code from the beginning

I learned that it is important to have a good structure in your code from the beginning. Ask Claude to create a folder structure and organise your code in a logical way. This will make it easier to find things later and will make your code more maintainable. I also learned that it is important to have a good naming convention for your files and variables. This will make it easier to understand what the code does later.

4

Claude can look at images

I found out that Claude can look at images. I used this to solve bugs in my platform game. I had tried to describe the bug to Claude, but he could not solve it. When I sent him a screenshot of the bug, he quickly found the problem and fixed it.

Claude Code Installation & Demo ▼

Quick walkthrough on how to install Claude Code and demonstrate its capabilities

Installation Steps:

  1. Install Node.js (if not already installed)
  2. Run: npm install -g @anthropics/Claude-code
  3. Run: Claude code to start
  4. Follow the setup wizard

Claude Code works great with any Integrated Development Environment (IDE) that has a terminal. Just run Claude, and you’re ready to go. Im using Visual Studio Code with the Claude extension.
